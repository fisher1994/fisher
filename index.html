首页 帮助
  • GIS软件工程
           遵循一般软件开发的工程化原理和方法，并按照GIS软件开发的特殊规律和要求，对GIS软件从（可行性研究、需求分析、总体设计、详细设计、软件编制、软件测试、软件维护）这几个阶段进行工程化规范的一门技术。
  • 软件工程是一门研究用工程化方法构建和维护有效的、实用的和高质量的软件的学科。它涉及程序设计语言、数据库、软件开发工具、系统平台、标准、设计模式等方面。
  • 地理信息系统（Geographic Information System或 Geo－Information system，GIS）有时又称为“地学信息系统”。它是一种特定的十分重要的空间信息系统。它是在计算机硬、软件系统支持下，对整个或部分地球表层（包括大气层）空间中的有关地理分布数据进行采集、储存、管理、运算、分析、显示和描述的技术系统。

课外超链接

  •       百度地图JavaScript API是一套由JavaScript语言编写的应用程序接口，支持PC端和移动端基于浏览器的地图应用开发。

  •       ArcGIS API for JavaScript可以将ArcGIS Server提供的地图资源和其它资源(ArcGIS Online)嵌入到Web应用中。

  •       ECharts是一个纯Javascript图表库，提供直观，生动，可交互，可个性化定制的数据可视化图表。

  •       MapGIS是中国地质大学开发的通用工具型地理信息系统软件，可对空间数据进行采集、存储、检索、分析和图形表示。
