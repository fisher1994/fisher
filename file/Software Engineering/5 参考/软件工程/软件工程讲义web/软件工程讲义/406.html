(8) 作用范围与控制范围： 模块的影响所及应处于以模块为根节点的子树中；

(9) 扇入和扇出： 模块被调用次数（扇入）越多，说明模块的复用性越好；模块调用其他模块（扇出）越少，模块的复杂性越小。
