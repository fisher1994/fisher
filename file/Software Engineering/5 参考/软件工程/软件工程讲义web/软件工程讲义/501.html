第五章 编程方法
5.1 编程的基本概念
    • 将详细设计的结果转换为计算机程序
    • 正确性（逻辑正确）和可读性（易于阅读）

5.2 程序设计语言

可以分为机器语言、汇编语言和高级语言（与详细设计时使用的伪码不同）。
    • FORTRAN (Formula Translator ), 1956
    • LISP (List Processor), 1958
    • COBOL（Common Business Oriented Language ), 1959
    • BASIC（Beginner's All-Purpose Symbolic Instruction Code ), 1964
    • PASCAL (法国数学家名字)，1971
    • C (BCPL: Basic Combined Programming Language => B => C => C++), 1973
    • ADA (英国诗人 Lord Byron 的女儿，曾经为 Charles Babbage 的差分机编写程序)
    • SMALLTALK （面向对象程序设计语言，OOP）1980

选择编程语言的一般准则：
    • 项目领域
    • 计算复杂性
    • 运行环境
    • 数据结构
    • 其他因素（性能要求、开发人员习惯）
