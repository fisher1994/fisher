3.4 数据词典

1．数据词典给出数据流图中的数据流名和文件名及其每一个组成部分（数据项）的定义。

2．数据词典条目的类型
  • 数据流条目：数据流的定义。通常是将数据流表示成数据项的和，数据项是指不能再分解的数据。

例如：取款单 = 帐号＋户名＋金额＋日期
  • 数据项条目：给出数据项的取值范围和限制条件。

例如：帐号＝00000－99999
  • 文件条目：文件条目和数据流条目相同，但是文件条目还可以指出文件的组织方式。
  • 数据项运算符号：

＋ 与运算

[|] 或运算

{} 重复，重复次数的上下限可在括号边上标出

() 可选项

例如：发票＝{货名＋数量＋单价＋[总价|金额]}

基本型（P47），扩充型（P48－P49）
