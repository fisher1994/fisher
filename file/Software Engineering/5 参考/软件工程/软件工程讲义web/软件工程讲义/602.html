6.4 白盒法

白盒法又称为逻辑覆盖法，即考虑测试用例对程序内部逻辑的覆盖程度。

按覆盖程度的大小可将白盒法分为下面几类：
    • 语句覆盖：选择测试用例使程序中每条语句至少被执行一次
    • 判定覆盖：选择测试用例使程序中每个分支至少被通过一次
      •
    • 条件覆盖：选择测试用例使判定中的每个条件获得两种结果
    • 判定/条件覆盖： 满足上面两条的要求
    • 条件组合覆盖：使每个判定中条件的各种组合至少出现一次
