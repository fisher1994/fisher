第七章 维护方法
7．1 基本概念

    在整个运行期中，为改正错误或新增功能而进行修改叫做软件（系统）维护。

    维护阶段的花费占整个软件生命期全部花费的67％。

7．2 维护种类

  • 完善性维护（扩充功能，提高性能）
  • 纠错性维护（处理系统故障或异常）
  • 适应性维护（适应环境的变化）
  • 预防性维护（改善软件的可行性）
