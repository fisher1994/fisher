第三章 需求分析和规格说明

3.2 结构化分析（SA）方法

需求分析的几种方法：

    • 结构化分析（SA）方法
    • 快速原型法
    • PSL/PSA，SREM

SA方法的基本原理：

    • 对复杂系统进行分解与抽象
    • 由顶往下、逐层分解、直到细节

SA方法的描述方式：

    • 分层数据流图，描述系统的分解过程
    • 数据词典，描述数据的组成情况
    • 小说明，描述系统的每个加工

SA方法的基本步骤：

    • 获得当前系统的具体模型
    • 抽象出当前系统的逻辑模型
    • 建立目标系统的逻辑模型（主要工作）
    • 必要的补充和完善
