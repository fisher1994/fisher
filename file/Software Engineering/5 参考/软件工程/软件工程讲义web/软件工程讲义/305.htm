第三章 需求分析和规格说明

3.3 数据流图（DFD）（续 2）

数据流图的改进：

1．正确性

    • 数据守恒定律：加工输出的数据应当来自于输入的数据
    • 文件的使用：读和写
    • 父图与子图平衡

2．易理解性：

    • 简化加工间的联系，即增加加工的独立性
    • 分解均匀性：同一张子图中的加工所处的层次位置相同
    • 用动词＋名词为加工命名

3．重新分解：

    • 将需要重新分解的图的所有子图拼起来
    • 分解这张拼图使各部分的联系最少，得到重新分解的子图
    • 将上面得到的每一部分画成一个加工，得到重新分解的父图
    • 为各张图中的所有加工重新编号命名
